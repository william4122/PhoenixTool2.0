Intro

            
This page explains how to use features of this project...

            

            
Infocyteapiquery
 --> is a Python module to access data from Infocyte Cloud SaaS platform. Detailed guide for the Python module is available here.
            Wiki section provides basic examples.
 --> If any issues or errors seen, please feel free to raise a ticket here.

            
Rules Tester
 --> is a project to test offline scan data against yaml rules and check if the rules are triggered.
 --> Currently this feature is still under design.

            
Rule Builder
 --> is used to generate new yaml rules.
        * Mandate: This field is must.
        * AND: Strict match the parameters.
        * OR: Optional match the parameters.
    Do's: - Use simple patterns (without escape characters).
          - Keep the rule as simple as possible.
    Don't: - No double quotes or double equals symbol.
 --> Once we have the parameters added in the text box, click on the Build Rule button
 --> Flow Diagram button can be used to check the rail road diagram.
 
Logalizer
 --> Reading the logs made easy by highliting the errors, warnings and success logs by simple colors.
 --> Upload the respective logs by button click and are ready to analyze.
Note: Incorrect logs uploaded using the button will cause the page reload. Max log size is 10MB.
Event Viewer Logs: Get-EventLog -LogName System -Message *infocyte* | Format-Table -AutoSize | Out-File 'C:\.....\Desktop\ic.txt'
Agent Logs: C:\Program Files\Infocyte\Agent\logs
Controller Logs: C:\Program Files\Infocyte\HUNT Controller\logs
GUI Logs: Task List --> Timeline
Tricks: Hover on the agent logs timestamp, UTC time will be converted to browser time
        Add 'tzone:Asia/Calcutta' keyword into the actual agent log file --> save --> upload, then UTC timestamp hover will be converted to Asia/Calcutta time.
